GBDI SlimTree Library Sample 1
==============================

This is a small example of use of the GBDI SlimTree Library. It shows how to use
the stDummyTree and stMetricTree and how to create the user layer.

Author: Marcos Rodrigues Vieira (mrvieira@icmc.sc.usp.br)
Author: Fabio Jun Takada Chino (chino@icmc.sc.usp.br)
Version: 1.0
Last Update: 07/29/02

Documentation Generation
========================

You may use Doxygen to generate the documentation of this example. Just run it in
this directory:
	doxygen

The generated documentation will be at doc directory under this directory tree.
